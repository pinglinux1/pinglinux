Text editing and processing

Share

What is Text editing?

All operating system comes with some basic test editor.Its just the word processor to write and modify the text.The most often used terminal mode text editor in linux is vi. Its updated versin is vim(vi improved).There are serveral other editors like nano,gVim and gEdit

Command mode and Input mode

vi initially opens in the command mode.The positioning commands only works in the command mode.Test editor will not accept any input in the command mode. You have to switch to the input mode in order for text editor to take the character you type and adds it to the file.To get into the input mode press character i and to exit input mode, press the escape (Esc) key.

Lets get started

To open the file, type vi newfilename
vi oldfilename (you can work on the existing file as well)
vi filename.txt (you can specify the extension as well)
once you open the file, you are in command mode, press i to start typing, When you are ready to save the file, press (ESC) then :wq or :x
Lets get into more details .....

Input Commands 

		a Append after cursor
		i Append before cursor
		o Open line below cursor
		O Open line above cursor
:r file Insert another file after current line

File Management commands

:w        saves the content to the file
:w filename    saves the contents to the file name specified
:q!       quits without saving the changes
:wq     saves and quits the editor
:x       saves and quits the editor
:sh     Exit the shell temporarily, can come back and catch where you left

Cursor Motions 

	    H Takes the cursor to the upper left corner
            M Takes the cursor to the middle line
	    L Takes the cursor to the lower left corner
  	    h Moves cursor back a character
	    j Moves cursor down a line
	    k Moves cursor up a line
	    l Moves cursor forward a character
	    ^ Moves cursor to the beggining of the line
            $ Moves the cursor to the end of the line
	    w one word forward
	    b one word backward

Deletion commands 

	     dd Delete the entire lines
             ndd Delete n number of lines
	     dw  Delete the word
             ndw Delete n number of words
             D Delete to end of line from current postion
	     x Delete a character

Pasteing or Recovering the deletions 

		p Paste after the cursor
		P paste before the cursor

Undo Commands 

	      u Undo last changes
	      U Undo all  changes

Arrangement commands 

                yy copy lines
		6yy yank 6 lines
		yw copy word
		J join lines

Regular Expression 

              ^ Matches beggining of the line
              $ Matches end of the line
	      . Matches any single character
	      .* Matches any character

Searching and Replacing 

             :/string search forward   
             :?string search backward
	     n repeat search forward
	     N repeat search backward
	     :s/pattern/replace   Replaces only the single word
             :%s/pattern/replaceword Replaces all the occurances

Motions 

        	:n Move the cursor to the nth line      
                 ctrl-d Scroll down half a screen
		 ctrl-u Scroll  up half a screen
		 ctrl-f  page forward
		 ctrl-b   Page backward

Move from old file to new file 

 	    Open the old file that you want to copy lines from
	    "a10yy will copy 10 lines and store in the buffer
	    :w write work buffer
	    :e newfilename
            "ap paste the text from a buffer

Line numbers 

             :set number
	     :set nonumber
	     :2,4yy copy from line 2 to 4
	     :2,3 co 105 copy line from 2 to to after line 105
	     :5,20 m $ Move lines from 5-20 to the end of the file
