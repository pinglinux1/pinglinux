What is Disk Partition ?

As the name implies, disk partition is simply partitioning or dividing the disk into more than one part.

Why is it important to partition a disk?

==> Disk partition has been followed for many years. Disk partitioning is important because of the following reasons:

  • Preserving the contents:
    • Lets say we have divided our disk into 3 partitions. If the first partition (partition contains sectors of hard disk) gets damaged, the remaining 2 partitions wont be affected.
  • Security:
    • If we have multiple partitions, if the data in one partition gets breached, the data in remaining partitions may still be safe.
  • Preventing system from unexpected errors:
    • Lets say, we have not partitioned our disk, and we only have one main partition. If a developer develops application that continuously generates error file, then whole system may freeze. In other words, rest of the application may not function. If we had partitioned and assigned separate partition for developer, then only that particular partition may be affected and applications in other partitions would function normally.

Understanding Partition

It is very important to know the basics of hard drive in order to understand the partition. Most of us are used to using windows and windows partition. The concept of partition in all operating system environment is same. However, the naming system in Linux is different than that of Windows. Instead of using letters like C, D, ... etc, we use /dev/sda, /dev/sdb, /dev/sd1, ..etc in Linux. Dev is an abbreviation for device and sd is an abbreviation for SCSI device.

 Note:Following are the differeces between hardlink and soft links:
Hard links have same inode numbers while softlinks do not
Hard links have actual file contents but softlinks are just pathway
Removing hard link just affects one file but removing original file affects the softlink.
Hard links cannot be created for the directory but softlinks can be created for directory.
