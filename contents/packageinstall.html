Package installation and Management

What is package?

Package is the software in linux,which are essentially archives containing a list of files.Packages can also contain scripts that run when the package is installed and removed.Package provides some kind of function on the system. There are various ways to get the package in the Redhat/CentOS

Yum command

yum is used to installed the package from the repository. it will first look into the repolist, which points to the repository locally or over the httpd or ftp. 

 yum installed httpd

Yum remove command is used to remove the package from the System.

 yum remove httpd

Yum list installed command is used to see all the package that has been installed on the system.

 yum  list installed

yum list will show all the installed, as well as availabe packages 

 yum  list

yum list will show all the availabe packages from the yum repo list

 yum  list available

yum list packagename will show the yum repo info it was downloaded from.

 yum  list httpd

yum update packagename will update the package and all the dependencies associated with it..

 yum  update httpd

yum check-update packagename will check if the update is available.

 yum  check-update httpd

yum-config-manager --enable repolist will enable the repolist.

  yum-config-manager --enable sensu

yum-config-manager --enable repolist will disable the repolist.

  yum-config-manager --disable sensu

Rpm

Its the individual package,which is archived as well. it doesnt take care of the dependencies accociated with the package

rpm -ivh packagename , will installed the package in the system 

rpm -ivh httpd-2.2.15-47.el6_7.1.x86_64.rpm

rpm -e packagename , will remove the package from the system 

rpm -e httpd-2.2.15-47.el6_7.1.x86_64

rpm -qa, will show all the rpm that is installed in the system

rpm -qa

rpm -qa | grep packagename , will check if rpm is installed 

rpm -qa | grep httpd-2.2.15-47.el6_7.1.x86_64

rpm -qpl packagename , will show all the files contained by rpm package and the location they are installed or will be installed/p> 

rpm -qpl httpd-2.2.15-47.el6_7.1.x86_64.rpm

rpm -qf locationoffile , will find outout which package a certain file belongs to 

[root@ip-172-31-31-37 conf]# rpm -qf /var/www/icons/world2.png
		  httpd-2.2.15-47.el6_7.1.x86_64

rpm --prefix=/location packagename , will install the into different location if it is relocatable. 

rpm -qa | grep httpd-2.2.15-47.el6_7.1.x86_64

Tar Balls

A tar ball is a (usually compressed) archive of files.Tar files usually comes with the extension .tar,.tar.gz,.tgz etc 

tar -xvf filename.tar 
 tar -xzvf filename.tar.gz

After you untar the file, get into the directory that has been untared, and bin folder mostly like contains the run script

Some time you might have to configured on your own

tar -xzf archive-name.tar.gz
cd archive-name
./configure
make all
make install

Specify the directory for file to get untared

tar -xvf filename.tar -C /target/directory
