Continue:

We can use continue in python to make the code not go below the loop if certain conditions meet. The code will continue to pick another entity or value and not go below the loop.

Continue Code sample: 

 
shop = ['sugar','milk','tea','chai','yougurt']
for item in shop:
    if item == 'milk':
        continue # it is telling if milk is seen continue for loop and do not go below. 
    print("buy " + item)
