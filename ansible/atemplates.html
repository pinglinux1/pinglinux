What is Templates ?

Templates are files that combine code, data, and text to produce a final output.it gets its input from users or from any stored resources

Templates Example

This template file will get its value from vars line.

vi index.j2
Hello {{ msg }} # here msg acts as a variable and gets it value from playbook ngix-playbook.yml

This will transfer the files into the remote host with the contents hello world.

vi nginx-playbook.yml 

---
- hosts: all
  vars:
  - msg: world

  tasks:
  - name: Upload the index.html to the pinglinux/html directory
    template: src=path/index.j2 dest=/var/www/pinglinux/html/ mode=677
